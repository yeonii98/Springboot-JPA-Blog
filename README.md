# SpringBoot 블로그 프로젝트
글 및 이미지 등록, 삭제 수정이 가능합니다.

## 🛠기술
* IDE : Spring Tool Suite 4
* DBMS : MySQL 5.7
* Frontend : BootStrap 4, Summernote, HTML, CSS
* Backend : Java, JavaScript, Spring Boot, Spring Security

## ⚙기능
* 게시글 CRUD
* 로그인 및 로그아웃
* 회원가입
